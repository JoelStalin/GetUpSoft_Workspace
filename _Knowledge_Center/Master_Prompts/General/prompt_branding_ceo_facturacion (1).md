# Prompt de Branding CEO para App de Gestión de Comprobantes y Facturas Electrónicas

Actúa como un CEO fundador, estratega de marca, experto en branding y estratega SEO para productos SaaS fintech/tax-tech en República Dominicana.

Tu misión es crear el branding completo de una app de gestión de comprobantes y facturas electrónicas, pensada para empresas, pymes, comercios, proveedores de servicios y emisores electrónicos que necesitan emitir, organizar, validar, consultar y dar seguimiento a sus documentos fiscales electrónicos.

Además, el servicio de comprobantes fiscales debe tener identidad de marca propia dentro del proyecto, con un nombre propio y un logotipo definido, aunque esté integrado dentro de la plataforma general.

La app debe transmitir:

- confianza
- cumplimiento
- modernidad
- seguridad
- agilidad
- control empresarial
- innovación local con proyección regional

## Contexto del producto

La plataforma está enfocada en:

- gestión de comprobantes fiscales electrónicos
- facturación electrónica
- emisión y control de e-CF
- trazabilidad documental
- validación de documentos
- control de clientes, ventas y reportes
- operación digital segura
- cumplimiento tributario
- experiencia simple para negocios reales

El producto debe sentirse:

- serio, premium y confiable
- tecnológico pero cercano
- corporativo sin ser frío
- dominicano / latino, pero con potencial internacional

## Quiero que generes

### 1. Diez nombres auténticos y memorables para el proyecto
- que suenen premium
- que sean fáciles de pronunciar
- que inspiren confianza
- que no parezcan genéricos
- que puedan funcionar como marca SaaS
- evita nombres demasiado largos o difíciles
- mezcla opciones modernas, ejecutivas y fintech

### 2. Para cada nombre, indica
- significado o lógica del nombre
- tono de marca que transmite
- por qué funciona para este tipo de app
- percepción que tendría un cliente al escucharlo

### 3. Luego elige los 3 mejores nombres y desarrolla para cada uno
- propuesta de valor de marca
- slogan
- personalidad de marca
- arquetipo de marca
- estilo visual recomendado
- colores sugeridos
- tipo de logo recomendado
- voz y tono de comunicación

### 4. Después selecciona 1 nombre ganador y crea
- narrativa de marca
- manifiesto corto
- pitch de marca de 1 párrafo
- descripción comercial para web
- bio corta para redes sociales
- tagline principal
- 3 variantes de tagline

### 5. Finalmente, crea una dirección creativa completa para el branding
- estilo minimalista / corporativo / fintech
- look & feel
- referencias visuales
- concepto de identidad verbal
- concepto de identidad gráfica
- recomendación de posicionamiento frente a competidores

### 6. Agrega la configuración SEO de la página y estrategia para mejorar visibilidad en Google
Incluye una propuesta completa de SEO para la web del proyecto, pensada para mejorar descubrimiento, indexación, posicionamiento orgánico y conversión.

Desarrolla:

- arquitectura SEO del sitio
- propuesta de páginas principales y páginas landing
- keywords principales, secundarias y long-tail
- intención de búsqueda del cliente ideal
- estructura de encabezados H1, H2 y H3 para home y páginas clave
- títulos SEO y meta descriptions sugeridos
- slugs recomendados
- estrategia de enlazado interno
- textos orientados a posicionamiento orgánico
- estrategia de contenido para blog
- propuesta de artículos para captar tráfico desde Google
- preguntas frecuentes con intención SEO
- estrategia local SEO para República Dominicana
- propuesta de autoridad de marca y confianza digital

### 7. Configuración técnica SEO que debe recomendar
Incluye recomendaciones concretas para que el proyecto nazca con una base técnica sólida en SEO:

- sitemap.xml
- robots.txt correctamente configurado
- etiquetas canonical
- metadatos Open Graph y Twitter Cards
- datos estructurados Schema.org en JSON-LD
- configuración para Google Search Console
- configuración para Google Business Profile si aplica
- optimización de velocidad y Core Web Vitals
- diseño mobile-first
- indexación correcta de páginas importantes
- exclusión de páginas irrelevantes o privadas
- estrategia para URLs limpias y semánticas
- imágenes optimizadas con alt text descriptivo
- jerarquía clara de contenido
- recomendación de tracking SEO y medición

### 8. Visibilidad en Google: enfoque estratégico
Explica cómo mejorar la visibilidad de la marca en Google a corto, mediano y largo plazo.

Incluye:
- acciones para aparecer mejor en búsquedas de marca
- acciones para posicionar búsquedas transaccionales
- acciones para posicionar búsquedas informativas
- estrategia de contenido evergreen
- estrategia de confianza y EEAT
- ideas para backlinks o alianzas legítimas
- estrategia de reputación digital
- recomendación de diferenciación frente a competidores
- plan de crecimiento orgánico de 3 a 6 meses


### 9. Identidad propia del servicio de comprobantes fiscales
Debes incluir una propuesta específica para que el servicio de comprobantes fiscales tenga identidad propia dentro del ecosistema del producto.

Incluye:

- nombre propio para el servicio de comprobantes fiscales
- lógica estratégica de ese nombre
- relación entre la marca principal y la submarca del servicio
- propuesta de logotipo para ese servicio
- estilo visual del logotipo
- versión textual del logotipo
- recomendación de uso del logo dentro de la app, web y materiales comerciales
- cómo se vería la arquitectura de marca si la plataforma tiene una marca madre y el servicio fiscal una identidad particular

Aclara si conviene una estrategia:
- monolítica
- endosada
- submarca
- arquitectura híbrida

## Requisitos SEO adicionales

- evita recomendaciones spam o black hat
- enfoca el SEO en buenas prácticas reales y sostenibles
- prioriza claridad, intención de búsqueda, autoridad y conversión
- adapta el SEO al sector de facturación electrónica, cumplimiento fiscal y software empresarial
- piensa como un estratega de crecimiento, branding y posicionamiento orgánico

## Restricciones

- No uses nombres cliché como: “eFactura App”, “Factura Fácil”, “Mi Comprobante”
- Evita nombres demasiado técnicos o aburridos
- Evita nombres difíciles de registrar o poco distintivos
- Prefiere nombres con fuerza de marca y potencial comercial
- Debe sentirse como una marca que un CEO presentaría con orgullo a inversionistas y grandes clientes
