# Backlog: Módulo de Cocina (KDS) para Restaurantes

## Descripción
Desarrollo de un módulo de cocina (Kitchen Display System) para optimizar el flujo de pedidos entre el salón y la cocina.

## Clientes / Fases
- [ ] **Chefalitas**: Implementación del módulo de cocina.
- [ ] **Mi Paladar**: Implementación del módulo de cocina.
- [ ] **Rickos Restauran**: Implementación del módulo de cocina.

## Funcionalidades Clave
- Visualización de pedidos en tiempo real.
- Gestión de estados de platos (Pendiente, En Preparación, Listo, Entregado).
- Alertas de tiempo de espera.
- Histórico de tiempos de preparación por plato.
- Integración con el sistema de pedidos actual.
