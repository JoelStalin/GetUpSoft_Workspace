A premium, high-fidelity UI mockup of an Odoo ERP dashboard for a Dominican enterprise. Dark mode, deep charcoal and navy background. The dashboard shows elegant data visualizations: a clean line chart for sales, a bar chart for inventory levels, a table for accounts payable, and a status badge for 'DGII Compliance'. The style is ultra-professional, using glassmorphism, thin lines, and a mint teal accent (#99F6E4). No cartoonish elements. Minimalist and architectural. Space Grotesk typography. Background #0F1115.
