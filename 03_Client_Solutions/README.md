# Client Solutions

This folder is the target architecture area for products and solutions built by GetUpSoft for external clients.

Existing client projects remain in their current paths until `migration_manifest.md` approves a controlled move.

## Known client solutions

- GalantesJewelry
- ChefAlitas
- EXO (if confirmed as client-specific implementation)

## Rules

- A Client Solution may consume Products.
- A Client Solution may consume Workers.
- A Client Solution may include client-specific adapters and configuration templates.
- A Client Solution must not contain production secrets.
- A Client Solution must have a Client Solution Card before migration.
