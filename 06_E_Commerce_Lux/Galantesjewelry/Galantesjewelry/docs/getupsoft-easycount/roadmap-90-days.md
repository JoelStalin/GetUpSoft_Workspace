# 90-Day Roadmap

## Weeks 1-2: Governance And Foundations

- Complete ADR-001 through ADR-004.
- Provision GitHub repositories and baseline CI.
- Confirm Cloudflare zones and DNS ownership.
- Complete hardcode/branding inventory.

## Weeks 3-4: Contracts And Environments

- Finalize OpenAPI v1 for the Odoo BFF.
- Create dev/staging/prod environment naming and variable matrix.
- Configure secrets per environment.
- Establish CORS and OAuth callback allowlists.

## Weeks 5-7: Product And Brand Implementation

- Publish GetUpSoft brand guide v1.
- Create corporate web information architecture.
- Implement EasyCount co-branding rules.
- Start platform scaffold in `easycount-platform`.

## Weeks 8-10: Integration And Quality

- Build Odoo JSON-2 BFF endpoints.
- Add E2E tests for customer, invoice, payment, and status flows.
- Add observability dashboard and alert rules.
- Define SLO/SLA v1.

## Weeks 11-12: Release Readiness

- Execute Go/No-Go gates.
- Run security baseline checks.
- Validate rollback procedures.
- Release first controlled staging demo.
