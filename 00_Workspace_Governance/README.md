# Workspace Governance

Esta carpeta centraliza el gobierno de arquitectura y clasificación del workspace empresarial de GetUpSoft.

## Contenido

- `directory_rules.md`
- `migration_manifest.md`
- `workspace_inventory.md`
- `dependency_map.md`

## Propósito

- Definir reglas de clasificación entre Product, Client Solution, Worker, ERP, Infrastructure y Knowledge.
- Registrar migraciones futuras antes de mover carpetas.
- Mantener un inventario inicial con riesgos y restricciones.
- Evitar reorganizaciones físicas inseguras en proyectos activos.
