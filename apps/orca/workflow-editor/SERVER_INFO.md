## Qué Puedes Hacer

### ✅ Agregar Nodos
1. Busca un nodo en el panel izquierdo
2. **HACES CLICK** en el nodo que quieres agregar
3. ¡Se agrega al canvas automáticamente!

**Ejemplo:**
- Escribe "trigger" → Haz click en Trigger
- Escribe "loop" → Haz click en Loop
- Escribe "ai" → Haz click en AI Prompt

|-------|--------|
| `Ctrl+Z` | Undo |

| `Ctrl+V` | Pegar nodos |
| `Ctrl+D` | Duplicar nodo seleccionado |
| `Ctrl+A` | Seleccionar todos los nodos |
| `Delete` / `Backspace` | Eliminar nodo/conexión seleccionados |
| `Ctrl+K` | Abrir búsqueda global |

## Nodos Disponibles (8 tipos)

1. **Trigger** (Rojo) - Inicia un workflow
2. **AI Prompt** (Púrpura) - Llama a un modelo de IA
3. **HTTP Request** (Cyan) - Hace peticiones HTTP
4. **Condition** (Naranja) - Rama condicional (2 salidas)
5. **Loop** (Verde) - Itera sobre una lista
6. **Set Variable** (Gris) - Almacena valores
7. **Execute** (Rojo) - Ejecuta comandos
8. **End** (Negro) - Finaliza el workflow
