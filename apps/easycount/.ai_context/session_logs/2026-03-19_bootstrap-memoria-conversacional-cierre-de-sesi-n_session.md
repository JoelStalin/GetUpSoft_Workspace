# Prompt de Branding CEO para App de Gestión de Comprobantes y Facturas Electrónicas

Actúa como un CEO fundador, estratega de marca y experto en branding para productos SaaS fintech/tax-tech en República Dominicana.

Tu misión es crear el branding completo de una app de gestión de comprobantes y facturas electrónicas, pensada para empresas, pymes, comercios, proveedores de servicios y emisores electrónicos que necesitan emitir, organizar, validar, consultar y dar seguimiento a sus documentos fiscales electrónicos.

La app debe transmitir:

- confianza
- cumplimiento
- modernidad
- seguridad
- agilidad
- control empresarial
- innovación local con proyección regional

## Contexto del producto

La plataforma está enfocada en:

- gestión de comprobantes fiscales electrónicos
- facturación electrónica
- emisión y control de e-CF
- trazabilidad documental
- validación de documentos
- control de clientes, ventas y reportes
- operación digital segura
- cumplimiento tributario
- experiencia simple para negocios reales

El producto debe sentirse:

- serio, premium y confiable
- tecnológico pero cercano
- corporativo sin ser frío
- dominicano / latino, pero con potencial internacional

## Quiero que generes

### 1. Diez nombres auténticos y memorables para el proyecto
- que suenen premium
- que sean fáciles de pronunciar
- que inspiren confianza
- que no parezcan genéricos
- que puedan funcionar como marca SaaS
- evita nombres demasiado largos o difíciles
- mezcla opciones modernas, ejecutivas y fintech

### 2. Para cada nombre, indica
- significado o lógica del nombre
- tono de marca que transmite
- por qué funciona para este tipo de app
- percepción que tendría un cliente al escucharlo

### 3. Luego elige los 3 mejores nombres y desarrolla para cada uno
- propuesta de valor de marca
- slogan
- personalidad de marca
- arquetipo de marca
- estilo visual recomendado
- colores sugeridos
- tipo de logo recomendado
- voz y tono de comunicación

### 4. Después selecciona 1 nombre ganador y crea
- narrativa de marca
- manifiesto corto
- pitch de marca de 1 párrafo
- descripción comercial para web
- bio corta para redes sociales
- tagline principal
- 3 variantes de tagline

### 5. Finalmente, crea una dirección creativa completa para el branding
- estilo minimalista / corporativo / fintech
- look & feel
- referencias visuales
- concepto de identidad verbal
- concepto de identidad gráfica
- recomendación de posicionamiento frente a competidores

## Restricciones

- No uses nombres cliché como: “eFactura App”, “Factura Fácil”, “Mi Comprobante”
- Evita nombres demasiado técnicos o aburridos
- Evita nombres difíciles de registrar o poco distintivos
- Prefiere nombres con fuerza de marca y potencial comercial
- Debe sentirse como una marca que un CEO presentaría con orgullo a inversionistas y grandes clientes

## Formato de salida

Organiza la respuesta así:

**A. Territorio de marca**
**B. 10 propuestas de nombre**
**C. Top 3 con desarrollo estratégico**
**D. Nombre ganador**
**E. Branding completo del ganador**
**F. Recomendación final como CEO**
```
