1. **Registrarse en SendGrid**

   ```

2. **Crear API Key**
   ```
   https://app.sendgrid.com

   → Copiar clave (aparece UNA SOLA VEZ)
   ```

3. **Actualizar .env.local**
   ```

   SMTP_PORT=587

   SMTP_SECURE=true
   SMTP_FROM=sistema@getupsoft.com.do
   ```

4. **Probir**

   python send_sendgrid_test.py

5. **Verificar en Gmail**
   ```
