# Decision: Prompt Maestro como contrato operativo DGII

Fecha: 2026-03-26
Decision: el prompt maestro de integracion DGII e-CF se adopta como contrato operativo para fases 0-9.
Razon: asegurar trazabilidad, calidad y evidencia verificable por fase.
Impacto: toda implementacion DGII debe mapearse a requisitos del prompt y validar gates.
