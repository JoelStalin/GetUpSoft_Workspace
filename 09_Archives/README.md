# 09_Archives — GetUpSoft Workspace

**Domain:** Archives
**ISO Reference:** ISO/IEC 12207:2017 §7.2.2 · ISO/IEC 27001:2022 A.8.10

This directory contains obsolete, replaced, or mothballed projects and code that is no longer actively maintained. Code here is preserved for historical reference and potential recovery of specific logic.

## Rules

- Do NOT delete content from this directory without human approval
- Content here is read-only — do not make changes expecting them to be deployed
- If you need to recover logic from an archived project, copy it out — do not modify it here
- All content here is tracked in git history (full recovery always available via `git log`)
